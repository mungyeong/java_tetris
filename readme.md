#자바 테트리스  

###기능
테트리스 도형 이동 방향키 (← ,→)<br>
테트리스 도형 한칸 다운 방향키 (↓)<br>
테트리스 도형 90도  방향키 (↑)<br>
테트리스 도형 밑으로 다운 키 및 게임 시작 (space bar)<br>
테트리스 게임 중단 (esc)<br>
테트리스 점수판<br>
테트리스 점수 JSON 파일 읽기 쓰기<br>
네트워크 연결(2인용)<br>

###메뉴 구성
메인 메뉴<br>
테트리스 보드판<br>
테트리스 점수 기록판<br>

### 점수 기준
줄당 10점<br>
4줄이상일 경우 줄당 15점<br>
<br>

### 공격 기준
4줄 이상
